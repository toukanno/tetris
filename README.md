# 🧩 Tetris

落ちてくるブロック（テトロミノ）を操作してラインを揃える定番パズルゲーム。

## 遊び方
- **←→** — ブロックを左右に移動
- **↑ / Z** — ブロックを回転
- **↓** — ソフトドロップ（ゆっくり落とす）
- **SPACE** — ハードドロップ（一気に落とす）
- **P** — 一時停止

## 特徴
- ゴーストピース表示（落下先をプレビュー）
- NEXTピース表示
- レベルアップで速度上昇
- 同時消し数ボーナス（テトリスで×8倍）
- ハイスコア記録

## 技術スタック
HTML5 Canvas / Vanilla JavaScript
